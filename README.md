### Creating development environment

- Pre-requisites:
  - [Docker](https://www.docker.com/) installed.
- Up docker development environment: `docker compose -f docker-compose.dev.yml up --build -d`
  - Create a **.env** file based on **.env.template** file.
- Now it's already possible to run the development version: `http://localhost:3000`
